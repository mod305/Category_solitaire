# Category Solitaire

A web-based solitaire game with categories, images, and dynamic balancing.

## How to Play
- Sort cards into the top slots by category.
- Stack cards in the columns (Tableau).
- Use the restart button to try a new random layout.

## Features
- **Mobile Support**: Optimized for touch and vertical screens.
- **Dynamic Balancing**: Turns calculated based on difficulty.
- **100+ Categories**: Randomized content every game.
